## Orphan Detection

A note is "orphan" (–∏–∑–æ–ª–∏—Ä–æ–≤–∞–Ω–Ω–∞—è) if:
- No incoming links from other notes
- No related notes in frontmatter
- Not listed in any MOC

Flag orphans for review:
```
<b>‚ö†Ô∏è –ò–∑–æ–ª–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –∑–∞–º–µ—Ç–∫–∏:</b>
```
